Markdown 段落没有特殊的格式，直接编写文字就好

字体
-
eg：
*斜体文本*   
_斜体文本_   
**粗体文本**   
__粗体文本__   
***粗斜体文本***   
___粗斜体文本___   
   
分割线
-
    *可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，同行内不能有其他东西。
    *可以在星号或是减号中间插入空格
eg：
***
* * *
*****
- - -
----------

下划线
-
    下划线可以通过 HTML 的 <u> 标签来实现
eg：
    <u>带下划线文本</u>

脚注
-
脚注是对文本的补充说明
Markdown 脚注的格式如下  
```   [^要注明的文本]```

eg:
创建脚注格式类似这样[^RUNOOB]   
[^RUNOOB]: 菜鸟教程 -- 学的不仅是技术，更是梦想！！！
